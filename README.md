# Telecom Customer Churn Analysis

## Project Overview
This project analyzes customer churn in the telecom industry using Power BI.
The goal is to identify key drivers of churn and provide actionable insights
to improve customer retention.

## Tools Used
- Power BI
- Excel / CSV
- GitHub

## Dataset
The dataset contains customer demographic information, service usage,
contract details, and churn status.

## Key Insights
- Higher churn observed among month-to-month contracts
- Customers with high monthly charges are more likely to churn
- Lack of long-term contracts correlates with churn risk


## How to Use
1. Download the `.pbix` file from the `powerbi` folder
2. Open it in Power BI Desktop
3. Refresh data if required
